How This Works

  1. You generated a quantum-safe key pair (XMSS)
  2. Your public key is stored on the server
  3. To login, the server sends you a unique nonce
  4. You sign the nonce with your private key
  5. Server verifies the signature using your public key
  6. If valid, you're authenticated (no password needed!)
